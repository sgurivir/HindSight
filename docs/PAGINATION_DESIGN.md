# Pagination and Copy All Button Design Document

## Overview

This document outlines the design for implementing two features in the HTML report generated by the report generation logic. The HTML template is embedded within Python files that generate the reports.

**Features:**
1. **Pagination**: Display issues 20 per page with navigation controls
2. **Copy All Button Enhancement**: Copy only filtered issues (respecting directory and severity selections)

## Affected Files

The HTML report template is embedded in the following Python files:

| File | Purpose | Report Type |
|------|---------|-------------|
| [`hindsight/report/report_generator.py`](hindsight/report/report_generator.py) | Main report generator with embedded HTML/CSS/JS template | Code Analysis, Trace Analysis |
| [`hindsight/analyzers/code_analyzer.py`](hindsight/analyzers/code_analyzer.py) | Code analyzer that calls report_generator | Code Analysis |
| [`hindsight/diff_analyzers/git_simple_diff_analyzer.py`](hindsight/diff_analyzers/git_simple_diff_analyzer.py) | Diff analyzer for git changes | Diff Analysis |

**Primary Change Location:** [`hindsight/report/report_generator.py`](hindsight/report/report_generator.py) - This file contains the `generate_html_report()` function (line 192) which has the complete HTML template with embedded CSS and JavaScript. All pagination and Copy All changes will be made to this embedded template.

The `code_analyzer.py` and `git_simple_diff_analyzer.py` call the report generator, so changes to the template in `report_generator.py` will automatically apply to reports generated by both analyzers.

## Feature 1: Pagination

### Requirements

1. Display 20 issues per page
2. Navigation buttons with page numbers in the top-right area
3. Pagination must honor:
   - User's severity filter selection (All Issues, Critical, High, Medium, Low)
   - User's directory selection in the left pane
4. Hide pagination controls when there are 20 or fewer issues

### Design

#### 1.2 Pagination Logic

Create new functions to handle pagination:

```javascript
// Calculate total pages based on filtered issues
function calculatePagination(filteredIssues) {
    paginationState.totalFilteredItems = filteredIssues.length;
    paginationState.totalPages = Math.ceil(filteredIssues.length / paginationState.itemsPerPage);
    
    // Reset to page 1 if current page exceeds total pages
    if (paginationState.currentPage > paginationState.totalPages) {
        paginationState.currentPage = 1;
    }
    
    // Ensure at least 1 page
    if (paginationState.totalPages < 1) {
        paginationState.totalPages = 1;
    }
}

// Get issues for current page
function getPagedIssues(filteredIssues) {
    const startIndex = (paginationState.currentPage - 1) * paginationState.itemsPerPage;
    const endIndex = startIndex + paginationState.itemsPerPage;
    return filteredIssues.slice(startIndex, endIndex);
}

// Navigate to specific page
function goToPage(pageNumber) {
    if (pageNumber >= 1 && pageNumber <= paginationState.totalPages) {
        paginationState.currentPage = pageNumber;
        applyFilters(); // Re-render with new page
    }
}

// Navigate to next page
function nextPage() {
    if (paginationState.currentPage < paginationState.totalPages) {
        goToPage(paginationState.currentPage + 1);
    }
}

// Navigate to previous page
function prevPage() {
    if (paginationState.currentPage > 1) {
        goToPage(paginationState.currentPage - 1);
    }
}
```

#### 1.4 Pagination Rendering Function

```javascript
function renderPaginationControls() {
    const container = document.getElementById('paginationContainer');
    const pagesContainer = document.getElementById('paginationPages');
    const prevBtn = document.getElementById('prevPageBtn');
    const nextBtn = document.getElementById('nextPageBtn');
    
    // Hide pagination if 20 or fewer items
    if (paginationState.totalFilteredItems <= paginationState.itemsPerPage) {
        container.style.display = 'none';
        return;
    }
    
    container.style.display = 'flex';
    
    // Enable/disable prev/next buttons
    prevBtn.disabled = paginationState.currentPage === 1;
    nextBtn.disabled = paginationState.currentPage === paginationState.totalPages;
    
    // Generate page numbers
    pagesContainer.innerHTML = '';
    
    const maxVisiblePages = 5;
    let startPage = Math.max(1, paginationState.currentPage - Math.floor(maxVisiblePages / 2));
    let endPage = Math.min(paginationState.totalPages, startPage + maxVisiblePages - 1);
    
    // Adjust start if we're near the end
    if (endPage - startPage < maxVisiblePages - 1) {
        startPage = Math.max(1, endPage - maxVisiblePages + 1);
    }
    
    // First page + ellipsis if needed
    if (startPage > 1) {
        pagesContainer.innerHTML += `<button class="pagination-page" onclick="goToPage(1)">1</button>`;
        if (startPage > 2) {
            pagesContainer.innerHTML += `<span class="pagination-ellipsis">...</span>`;
        }
    }
    
    // Page numbers
    for (let i = startPage; i <= endPage; i++) {
        const activeClass = i === paginationState.currentPage ? 'active' : '';
        pagesContainer.innerHTML += `<button class="pagination-page ${activeClass}" onclick="goToPage(${i})">${i}</button>`;
    }
    
    // Last page + ellipsis if needed
    if (endPage < paginationState.totalPages) {
        if (endPage < paginationState.totalPages - 1) {
            pagesContainer.innerHTML += `<span class="pagination-ellipsis">...</span>`;
        }
        pagesContainer.innerHTML += `<button class="pagination-page" onclick="goToPage(${paginationState.totalPages})">${paginationState.totalPages}</button>`;
    }
}
```

## Feature 2: Copy All Button Enhancement

### Requirements

1. "Copy All" button should copy issues matching current filters:
   - **Always respects** the selected directory (from left pane)
   - **Respects** the selected severity filter (Critical, High, Medium, Low) when a specific severity is selected
2. If "All Issues" severity tab is selected, copy all issues for the selected directory (or all directories if "All Issues" is selected in the left pane)
3. The button should **always honor the directory selection** - never copy issues from directories not currently selected

### Behavior Matrix

| Directory Selection | Severity Selection | Copy All Behavior |
|---------------------|-------------------|-------------------|
| All Issues | All Issues | Copy all issues in report |
| All Issues | Critical | Copy all Critical issues |
| Mac | All Issues | Copy all issues in Mac directory |
| Mac | High | Copy only High severity issues in Mac directory |
| SafariShared | Medium | Copy only Medium severity issues in SafariShared directory |

### Design

#### 2.1 Current Implementation Analysis

The current [`copyAllIssues()`](hindsight/report/report_generator.py:1202) function at line 1202 already uses `currentlyDisplayedIssues`:

```javascript
function copyAllIssues() {
    const allIssues = currentlyDisplayedIssues.length > 0 ? currentlyDisplayedIssues : issuesData.issues;
    // ...
}
```

However, with pagination, `currentlyDisplayedIssues` will only contain the current page's issues. We need to track ALL filtered issues (not just the current page).

#### 2.2 Solution: Track All Filtered Issues

Add a new variable to track all filtered issues (across all pages):

```javascript
// Store currently displayed issues for copy functionality
let currentlyDisplayedIssues = [];  // Issues on current page (for individual copy)
let allFilteredIssues = [];          // All issues matching current filters (for Copy All)
```

#### 2.3 Modified `applyFilters()` Function

Update to track both paged and all filtered issues:

```javascript
function applyFilters() {
    let filteredIssues = issuesData.issues;

    // Apply severity filter
    if (currentFilters.severity !== 'all') {
        filteredIssues = filteredIssues.filter(issue => issue.severity === currentFilters.severity);
    }

            const issueDir = getIssueDirectory(issue.file_path || issue.file);
            return issueDir === currentFilters.directory || issueDir.startsWith(currentFilters.directory + '/');
        });

    allFilteredIssues = filteredIssues;

    renderIssues(pagedIssues);
    
    // Update pagination controls
    renderPaginationControls();
}
```

## Edge Cases

1. **Empty filter results**: Show "No issues found" message, hide pagination
2. **Single page**: Hide pagination controls entirely
3. **Last page with fewer items**: Display remaining items (may be < 20)
4. **Filter change mid-pagination**: Reset to page 1
5. **Copy All with no matching issues**: Show appropriate alert message
